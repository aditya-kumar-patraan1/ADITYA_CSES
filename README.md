## 👨‍💻 About

This repository contains my **C++ solutions** to the CSES Problem Set organized by topic. CSES (Code Submission Evaluation System) is a competitive programming platform maintained by the University of Helsinki, featuring a curated set of algorithmic problems.

- 🔗 **Official Site:** [https://cses.fi/problemset](https://cses.fi/problemset)
- 🧑‍🎓 **Author:** [Aditya Kumar Patraan](https://github.com/aditya-kumar-patraan1)
- 💻 **Language:** C++

## 🛠️ How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aditya-kumar-patraan1/ADITYA_CSES.git
   cd ADITYA_CSES
   ```

2. **Compile any solution:**
   ```bash
   g++ -O2 -o solution solution.cpp
   ./solution
   ```

3. **Or use the competitive programming extension (CPH) in VS Code** which is already configured in this repo! ⚡
